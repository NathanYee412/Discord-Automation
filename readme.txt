# Discord-Automation
Discord automation using Selenium

# Requirements 
# 1. Google Chrome 
# 2. Python

# Steps
# 0. Pull github repository 
# 1. pip install selenium 
# 2. Place chromedriver.exe into C/:Program Files(x86)/
# 3. Open discord on web browser and go to the channel and text channel you want to run the bot in
# 4. Place that link into the discord link variable 
# 5. run selbot.py
